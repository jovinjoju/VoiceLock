# VoiceLock

A simple Android application that allows users to lock their phone screen using a specific voice command: "Lock Screen".

## Features
- **Hands-free Locking:** Uses voice recognition to trigger the system lock screen.
- **Background Service:** Runs in the background using an Accessibility Service to ensure it is always ready to listen.

## Setup
1. Clone this repository to your computer.
2. Open the project in Android Studio.
3. **Important:** After installation, you must go to **Settings > Accessibility** and manually enable the VoiceLock service.
